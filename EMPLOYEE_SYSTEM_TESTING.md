# دليل اختبار نظام إدارة الموظفين - TabibiQ

## نظرة عامة
هذا الدليل يوضح كيفية اختبار نظام إدارة الموظفين الجديد الذي تم إضافته إلى منصة TabibiQ.

## المتطلبات الأساسية
- Node.js مثبت على النظام
- الخادم يعمل على المنفذ 5000
- قاعدة البيانات MongoDB متصلة
- حزمة axios مثبتة

## تثبيت المتطلبات
```bash
cd backend-iq
npm install axios
```

## تشغيل الاختبارات

### 1. اختبار سريع
```bash
node test-employee-system.js
```

### 2. اختبار منفصل
```javascript
const { testEmployeeSystem, testPointsSystem } = require('./test-employee-system');

// اختبار نظام الموظفين
await testEmployeeSystem();

// اختبار نظام النقاط
await testPointsSystem();
```

## ما يتم اختباره

### ✅ اختبارات نظام الموظفين
1. **إضافة موظف جديد** - إنشاء موظف مع جميع البيانات المطلوبة
2. **جلب الموظفين** - استرجاع قائمة الموظفين للطبيب
3. **البحث عن موظف** - البحث في الموظفين حسب الاسم أو رقم الهاتف
4. **جلب الإحصائيات** - استرجاع إحصائيات الموظف (أسبوعي/شهري/سنوي)
5. **حذف موظف** - إزالة موظف من النظام
6. **التحقق من الحذف** - التأكد من إزالة الموظف بنجاح

### ✅ اختبارات نظام النقاط
1. **إضافة نقاط** - إضافة نقاط للموظف مع السبب والتاريخ
2. **جلب النقاط** - استرجاع جميع نقاط الموظف
3. **تنظيف البيانات** - حذف بيانات الاختبار

## رسائل الخطأ المحتملة

### خطأ 404 - الخادم غير متصل
```
❌ حدث خطأ أثناء الاختبار: Request failed with status code 404
💡 تأكد من أن الخادم يعمل على المنفذ 5000
```

### خطأ 500 - خطأ في قاعدة البيانات
```
❌ حدث خطأ أثناء الاختبار: Request failed with status code 500
💡 تأكد من أن قاعدة البيانات متصلة وأن النماذج تم إنشاؤها
```

### خطأ في الاتصال
```
❌ حدث خطأ أثناء الاختبار: connect ECONNREFUSED 127.0.0.1:5000
💡 تأكد من تشغيل الخادم
```

## الميزات المتاحة للاختبار

### 🔧 إدارة الموظفين
- إضافة موظف جديد مع جميع البيانات
- عرض قائمة الموظفين
- حذف موظف
- البحث في الموظفين

### 📊 الإحصائيات
- إحصائيات أسبوعية
- إحصائيات شهرية  
- إحصائيات سنوية
- عدد المواعيد والحضور
- إجمالي النقاط ومتوسطها

### 📱 التحقق من صحة البيانات
- التحقق من صحة رقم الهاتف العراقي
- التحقق من البيانات المطلوبة
- رسائل خطأ واضحة

## استكشاف الأخطاء

### المشكلة: لا تظهر النافذة
**الحل:** تأكد من أن `EmployeeManagement` تم استيراده في `DoctorDashboard.js`

### المشكلة: لا يتم حفظ الموظف
**الحل:** تأكد من أن الخادم يعمل وأن API endpoint `/api/employees` متاح

### المشكلة: لا تظهر الإحصائيات
**الحل:** تأكد من أن endpoint `/api/employees/:id/stats` يعمل

### المشكلة: خطأ في الترجمة
**الحل:** تأكد من إضافة مفاتيح الترجمة في `translation.json`

## ملاحظات مهمة
- تأكد من أن جميع النماذج (Employee, Points, EmployeeStats) تم إنشاؤها في قاعدة البيانات
- تأكد من أن middleware المصادقة يعمل بشكل صحيح
- تأكد من أن جميع API endpoints متاحة ومحمية
